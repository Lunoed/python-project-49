**Installing**

In terminal input commands:

- Install packages:

```bash
make install
```
- Build dist:

```bash
make build
```

- Istall project in your system:

```bash
make package-istall
```

### Games and their descriptions:

**Parity check**

The game is as follows: the user is shown a random number and must answer “yes” if the number is even, or “no” if it is odd.

[![asciicast](https://asciinema.org/a/819869.svg)](https://asciinema.org/a/819869)

**Calculator**

The user is shown a random mathematical expression, for example 16 + 99, which must be calculated and the correct answer written down.

[![asciicast](https://asciinema.org/a/819867.svg)](https://asciinema.org/a/819867)

**Is a number prime?**

The user must answer “yes” if the number is prime and “no” if it is not.

[![asciicast](https://asciinema.org/a/819910.svg)](https://asciinema.org/a/819910)

**Arithmetic Progression**

The user is shown a series of numbers that form an arithmetic progression, one of which is replaced by two dots. The user must identify this number.

[![asciicast](https://asciinema.org/a/819911.svg)](https://asciinema.org/a/819911)

**Greatest Common Divisor(GCD)**

The essence of the game is as follows: the user is shown two random numbers, for example, 25 50. The user must calculate and enter the greatest common divisor of these numbers.

[![asciicast](https://asciinema.org/a/819870.svg)](https://asciinema.org/a/819870)
